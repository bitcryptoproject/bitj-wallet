Λίγα λόγια για την ασφάλεια των Bits σας:

  • α Bits στο το Bit Wallet είναι αποθηκευμένα στη συσκευή. Αν χάσετε τη συσκευή σας, θα χάσετε τα Bits σας. Ακριβώς όπως τα χρήματα στο πραγματικό πορτοφόλι σας.
  • Πριν από την απεγκατάσταση του το Bit Wallet (εκκαθάριση δεδομένα εφαρμογής), θα πρέπει να μεταφέρετε Bits σας σε άλλο λογαριασμό. Τυχόν υπόλοιπα Bits θα χαθούν.
  • Οι συναλλαγές δεν μπορoούν να αναιρεθούν. Εάν στείλετε Bits στο κενό, δεν υπάρχει τρόπος να τα πάρουν πίσω, εκτός αν γνωρίζετε τον παραλήπτη και είναι πρόθυμος να τα στείλει πίσω.
  • Κρατήστε ασφαλή την Android συσκευή σας! Να εγκαταστάτε μόνο εφαρμογές που εμπιστεύεστε πλήρως. Κακόβουλες εφαρμογές θα μπορούσε να προσπαθήσουν να κλέψουν το πορτοφόλι σας.
  • Κρατήστε τον κίνδυνο χαμηλά! Χρησιμοποιήστε το Bit Wallet μόνο με μικρά ποσά για ημερήσια χρήση.
